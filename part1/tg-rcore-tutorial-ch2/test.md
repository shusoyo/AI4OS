# 六、终极理解题（操作系统视角）

19️⃣ 假设某个用户程序：

* 发生非法访存
* 触发 Trap

为什么内核只会**杀死这个用户程序**，而不会崩溃？

trap 机制给了 os 更强大的容错能力，内核不会收到 app 的影响。
